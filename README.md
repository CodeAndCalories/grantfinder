# GrantFinder

GrantFinder is a searchable database of government grants for startups, businesses, nonprofits, and researchers.

## Features
- Search grants by keyword
- Filter by industry, location, funding amount, and deadline
- Sort by newest, deadline soon, or highest funding
- Detailed grant pages with eligibility and application links

## Tech Stack
- Next.js
- TypeScript
- Local JSON dataset (100 grants)

## Run locally

npm install  
npm run dev

Open http://localhost:3000

## Project Structure

app/ – pages and UI  
data/ – grant dataset  
lib/ – data utilities
